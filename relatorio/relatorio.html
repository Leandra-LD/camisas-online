1. Introdução

Este relatório descreve o desenvolvimento do projeto Camisas Online, um site de comércio eletrônico voltado para a venda de camisas. O trabalho foi realizado como avaliação prática da disciplina Desenvolvimento Web da Universidade Estácio de Sá, sob orientação do Professor Paulo Cruz.

O objetivo principal foi aplicar na prática os conceitos de desenvolvimento web, criando um site funcional, responsivo e organizado, utilizando as linguagens HTML5, CSS3, JavaScript e PHP.

O site simula uma loja virtual completa, com catálogo de produtos, galeria de fotos, carrinho de compras com persistência via localStorage, formulário de contato com validação client-side e relatório acadêmico.

2. Tecnologias Utilizadas

HTML5

Estrutura semântica das páginas com tags modernas como <header>, <nav>, <main>, <section>, <footer>.

CSS3

Estilização completa com Flexbox, Grid Layout, variáveis CSS, animações e design responsivo (media queries).

JavaScript

Interatividade: carrinho com localStorage, filtros de produtos, galeria lightbox e validação de formulários.

PHP

Linguagem de back-end utilizada para processamento de formulários, geração dinâmica de páginas e futura integração com banco de dados.

4. Páginas Desenvolvidas

index.html — Início

Página principal com hero banner, seção de vantagens, categorias, produtos em destaque e banner de oferta.

produtos.html — Catálogo

Listagem de 11 produtos com filtros por categoria, tamanho, preço e ordenação. Suporta busca global via sessionStorage.

galeria.html — Galeria

11 fotos organizadas em grid com filtro por categoria (Casual, Social, Polo, Estampada, Esporte) e lightbox para ampliar as imagens.

favoritos.html — Favoritos

Lista de produtos salvos pelo usuário, persistida via localStorage. Permite adicionar e remover favoritos com atualização do badge no header.

carrinho.html — Carrinho

Gerenciamento de itens com controle de quantidade, remoção, sistema de cupons e finalização de pedido.

contato.html — Contato

Formulário com validação JavaScript client-side. Inclui máscara de telefone e feedback visual.

sobre.html — Sobre Nós

Apresentação da empresa fictícia com números, valores institucionais e equipe com fotos.

5. Funcionalidades Implementadas

HTML Estrutura

  • Estrutura semântica com <header>, <nav>, <main>, <section>, <article>, <footer>
  • Meta tags de responsividade e charset UTF-8
  • Barra acadêmica em todas as páginas (Estácio, disciplina, professor, aluna, RA)
  • Formulários com atributos HTML5 (required, type, placeholder, pattern)
  • Links de navegação interligando todas as páginas

CSS Estilização

  • Sistema de variáveis CSS (custom properties) para paleta de cores consistente
  • Layout responsivo com CSS Grid e Flexbox
  • Media queries para mobile (480px, 720px, 960px)
  • Animações e transições suaves (hover, toast, lightbox)
  • Design moderno com cards, sombras, gradientes e bordas arredondadas
  • Navbar com indicador da página ativa

JavaScript Interatividade

  • Carrinho de compras persistido via localStorage
  • Filtros de produtos por categoria, tamanho, preço e ordenação
  • Busca global persistida via sessionStorage entre páginas
  • Galeria lightbox com navegação por teclado (setas e ESC)
  • Sistema de cupons de desconto (ESTACIO10, WEB20, PROMO15)
  • Validação de formulário no client-side com mensagens de erro
  • Notificações toast animadas
  • Máscara de telefone em tempo real
  • Contador dinâmico do carrinho no header
  • Toggles de visualização em grade/lista

PHP Back-end

  • Processamento server-side do formulário de contato (envio de e-mail via mail())
  • Geração dinâmica do catálogo de produtos a partir de banco de dados
  • Validação server-side dos dados enviados pelo usuário
  • Gerenciamento de sessões para controle de login e carrinho persistente
  • Integração com banco de dados MySQL para armazenamento de pedidos e clientes

6. Catálogo de Produtos

Os produtos são gerenciados via array JavaScript no arquivo js/main.js, simulando um banco de dados. Cada produto contém:

CampoTipoDescrição
idNumberIdentificador único do produto
nomeStringNome completo do produto
categoriaStringcasual | social | polo | estampada | esporte
precoNumberPreço de venda atual (R$)
precoDeNumberPreço original (riscado)
imagemStringNome do arquivo de imagem local (pasta images/)
avaliacaoNumber (0–5)Nota média das avaliações
avaliacoesNumberQuantidade de avaliações recebidas
tagStringoferta | novo | destaque
tamanhosArrayTamanhos disponíveis (PP, P, M, G, GG)
descricaoStringDescrição curta do produto

Ao todo são 11 produtos cadastrados, distribuídos nas 5 categorias disponíveis: Casual, Social, Polo, Estampada e Esporte.
